# FoodShare Pakistan - "Save Food, Feed People"

A modern, responsive, and professional web application dedicated to reducing food waste by connecting food donors with NGOs and volunteers across Pakistan.

## Technology Stack
- **Frontend**: HTML5, CSS3, Bootstrap 5, JavaScript
- **Backend**: Core PHP
- **Database**: MySQL
- **APIs**: Google Maps (Mock), Email Notifications (Mock)

## Core Features
- **Donor Portal**: Post food donations, manage history, view freshness scores, and earn reward points.
- **NGO Portal**: Browse available food, request donations, and track delivery status.
- **Volunteer Portal**: Claim pickup tasks, manage delivery routes, and mark tasks as completed.
- **Admin Panel**: User management, NGO verification, donation monitoring, and impact statistics.
- **Advanced Systems**: 
  - Environmental Impact Calculator (CO2 & Water saved)
  - Food Freshness Scoring System
  - Donor Achievement & Badge System

## Installation
1. Import `database/schema.sql` into your MySQL server.
2. Update `includes/config.php` with your database credentials.
3. Place the project in your local server directory (e.g., htdocs).
4. Access the platform via `http://localhost/foodshare-pakistan/`.

## Folder Structure
- `admin/`: Administrator dashboard and management tools.
- `donor/`: Food donor dashboard and donation tools.
- `ngo/`: NGO dashboard and food requesting tools.
- `volunteer/`: Volunteer dashboard and delivery tracking tools.
- `assets/`: CSS, JS, and image files.
- `includes/`: Common components (header, footer, config).
- `uploads/`: Directory for food and profile images.
- `database/`: SQL schema files.
